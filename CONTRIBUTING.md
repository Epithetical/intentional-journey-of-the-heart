# Contributing to Intentional Journey of the Heart

Thank you for your interest in contributing to this project. This work is licensed
under CC BY-NC-SA 4.0, and all contributions fall under the same license.

## Types of Contributions

### Corrections and Typos
If you find an error, open an Issue using the **Correction** template. Include the
page where the issue appears and your suggested fix.

### Response Papers
If you want to engage with a specific exploration, propose your response paper by
opening an Issue with the **Contribution Proposal** template. Once discussed,
fork the repo, write your paper in `docs/community/responses/`, and open a PR.

### Case Studies
Applied case studies go in `docs/community/case-studies/`. Same process: propose
via Issue, then fork and PR.

### Extensions
Building on an open trail from Volume 3? Propose it first, then submit via PR.

## Content Standards

All contributed content must include YAML front matter:

```yaml
---
title: "Your Title"
author: "Your Name"
date: 2026-03-29
tags: [relevant, tags]
---
```

Response papers must cite the specific explorations they engage with using
relative markdown links.

## Review Process

All pull requests are reviewed by John G. Tittle and JD Tittle. We aim to
review submissions within two weeks. Please be patient — thoughtful theological
review takes time.

Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before participating.
