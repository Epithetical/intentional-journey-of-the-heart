# How to Use This Site

## Navigation

The **tabs at the top** let you jump between major sections: Introduction, each Volume, Formation Documents, Community, and Resources.

The **sidebar on the left** shows the full structure of whichever section you're in. Click any exploration to read it. Your current location is highlighted.

**Breadcrumbs** at the top of each page show your path (e.g., Home > Volume 2 > Exploration 3). Click any breadcrumb to jump back up.

## Search

The **search bar** at the top searches across all documents. Type any concept (e.g., "resonance", "knots", "faith") to find every page where it appears.

## Reading Order

The volumes build on each other. Volume 1 establishes the core laws, Volume 2 applies them, Volume 3 attempts quantitative modeling, and Volume 4 addresses testing. Volume 5 is the annotated reference library. See the [Reading Guide](../reading-guide.md) for suggested paths.

## Dark Mode

Toggle between light and dark mode using the sun/moon icon in the header.

## On Mobile

The sidebar collapses into a hamburger menu. Tap it to navigate. All features work on mobile.
