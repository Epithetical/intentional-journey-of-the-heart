# Community Contributions

This project is designed for collaboration. The explorations in these volumes are deliberately left with open trails and unresolved questions — invitations for others to pick up and extend.

## How to Contribute

**Response Papers** — Engage with a specific exploration. Agree, disagree, extend, or challenge the claims. Place your paper in `docs/community/responses/`.

**Case Studies** — Apply the framework to a real situation: a small group, a ministry context, a personal journey. Place case studies in `docs/community/case-studies/`.

**Extensions** — Volume 3 ends with a list of Open Trails that the author could not yet answer. Pick one up.

## Process

1. Open a GitHub Issue describing what you want to contribute
2. Fork the repository and create a branch
3. Write your contribution with YAML front matter (title, author, date, tags)
4. Open a Pull Request
5. John G. Tittle and JD Tittle will review

See [CONTRIBUTING.md](https://github.com/Epithetical/intentional-journey-of-the-heart/blob/main/CONTRIBUTING.md) for full details.
